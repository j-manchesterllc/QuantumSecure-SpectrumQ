# SpectrumQ

SpectrumQ is a research-oriented secure messaging platform exploring post-quantum cryptography, metadata resistance, and decentralized transport.

The project combines hybrid post-quantum key exchange, forward secrecy, and onion-routed communications into a unified architecture.

## Core Goals

- Post-quantum secure key exchange (PQXDH)
- Forward secrecy via double/triple ratchet design
- Metadata resistance and traffic analysis mitigation
- Modular transport layer (including Tor-compatible routing)
- Identity and contact abstraction layer

## Status

Early-stage protocol design and architecture definition (v0.1 foundation phase).

## Warning

This project is experimental and not yet suitable for production or real-world secure communication.
